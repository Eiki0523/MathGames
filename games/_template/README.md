# `_template` の役割

新しいゲームを追加するときの標準形です。

- `index.html`
  - 共通UIの骨組み
- `style.css`
  - そのゲーム固有の見た目
- `game.js`
  - 進行制御
- `config.js`
  - タイトル、モード、点数、解放条件、BGM設定
- `theme.js`
  - 色、文言、敵の見た目
- `problems.js`
  - 問題生成
- `input/`
  - 入力方式ごとのアダプタ

# 新しいゲームを作る手順

1. `_template` をゲーム用フォルダとして複製する
2. `config.js` でタイトル、モード、点数、BGMを変える
3. `theme.js` で色と敵デザインを変える
4. `problems.js` で出題内容を変える
5. 入力方式がボタン以外なら `input/` に新しいアダプタを追加する

# 設計ルール

- 画面の流れは共通
  - モード選択
  - 3秒カウントダウン
  - プレイ
  - リザルト
- 共通ロジックは `shared/` を使う
- ゲーム固有ロジックは各ゲームフォルダに置く
- BGM は `generated` と `file` の両対応
- 追加の入力方式は `input` フォルダで差し替える
